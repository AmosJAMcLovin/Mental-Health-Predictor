Mental Health Versus Suicide: Predicting Suicide Rates from Mental Health Risk Factors

Machine Learning

1. Introduction

This project used various machine learning models, trained and tested the data to see the models ability to predict whether or not an individual will seek treatment for a mental health condition. We used the linear regression, logistic regression, decision tree, random forest, support vector machine(SVM) models and natural language processing (NLP) algorithm. We note differences in accuracy (and thus, effectiveness), the models has toward the entire data set. We conclude that the linear regression, logistic regression, and support vector machine(SVM) models were both effective in classifying Although we used models to predict our outcome, we noticed similar reapearing patterns that we knew would serve of importance when predicting indivduals attitude toward mental health treatment. We used a natural language processing (NLP) algorithm in order to tokenize and classfy text from the comment section in order to determine which mental health topic revealed the most.
